# Docweaver

An easy-to-use product documentation package in Golang.

Docweaver is suitable for product documentation and/or knowledge bases. Converts folder(s) of .md files into full-bread complete documentation. 
This package is without UI elements and templates, all freedom regarding final presentation is given to the end-user.

PHP/Laravel version is available [here](https://github.com/reliqarts/laravel-docweaver).
